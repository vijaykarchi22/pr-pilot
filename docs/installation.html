Installing PR Pilot

PR Pilot is distributed through the JetBrains Marketplace — the official plugin repository built into all JetBrains IDEs. You can also install it manually from a ZIP file if you are working in an air-gapped environment.

Install from JetBrains Marketplace

This is the recommended installation method. The Marketplace handles all updates automatically.

  1. Open IntelliJ IDEA and go to Settings / Preferences (⌘, on macOS, Ctrl+Alt+S on Windows/Linux).
  2. Navigate to Plugins in the left sidebar.
  3. Click the Marketplace tab at the top.
  4. Search for PR Pilot.
  5. Click Install next to the PR Pilot entry.
  6. Accept the privacy agreement when prompted.
  7. Click Restart IDE to activate the plugin.

Tip: You can also install directly from the browser by visiting the PR Pilot Marketplace page and clicking Get.

Install from ZIP File

Use this method in air-gapped environments or if you want to pin to a specific version.

  1. Download the latest pr-pilot-<version>.zip from the GitHub Releases page.
  2. Open Settings → Plugins.
  3. Click the ⚙ gear icon → Install Plugin from Disk…
  4. Select the downloaded .zip file.
  5. Click Restart IDE.

Note: Plugins installed from disk do not receive automatic updates. You must repeat this process to upgrade.

Build from Source

If you want to contribute or test the latest changes:

# Clone the repository
git clone https://github.com/vitiquest/pr-pilot.git
cd pr-pilot

# Run the plugin in a sandboxed IDE instance
./gradlew runIde

# Build a distributable ZIP
./gradlew buildPlugin
# Output: build/distributions/pr-pilot-<version>.zip

See the build_instructions.txt file for the full build-from-source guide.

First Run

After restarting IntelliJ, PR Pilot is ready to use:

  1. Tool window: A new PR Pilot panel appears on the right side of the IDE. Click it to open the pull request browser.
  2. Skill files seeded: When you open a Git project for the first time, PR Pilot automatically creates .idea/pr-pilot/skills/ with three default Markdown files. See the Skills guide.
  3. Configure credentials: Go to Settings → PR Pilot to connect your GitHub or Bitbucket account and choose an AI backend. See the Configuration guide.

Pro tip: The PR Pilot tool window docks to the right side by default. You can drag it to the left, bottom, or float it as a separate window — just like any other IntelliJ tool window.

Uninstalling

  1. Go to Settings → Plugins → Installed.
  2. Find PR Pilot and click the three-dot menu → Uninstall.
  3. Restart the IDE.

PR Pilot stores its settings in <IDE config dir>/options/PRPilot.xml. This file is not deleted on uninstall. You can safely remove it manually if you want a clean slate.

Skill files in .idea/pr-pilot/skills/ are part of your project and are never touched during uninstall.
